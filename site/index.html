How it works

Three steps. No accounts, no servers, no subscriptions.

1
🔌

Open the plugin in Figma

Install from the Figma Community and run it from Plugins menu — works in any Figma file.

2
📂

Upload your .pen file

Drag and drop or browse for your Pencil.dev design file. No upload to any server — everything stays local.

3
✓

Pick screens and import

Choose which screens to import. The plugin creates fully editable Figma frames on a dedicated page.

What gets imported

Built specifically to handle the edge cases other importers miss.

⚡

Full Auto Layout

Pencil's flexbox layout — horizontal, vertical, wrap, gap, padding, justify, align — maps directly to Figma Auto Layout.

🧩

Component Instances

Reusable nodes become real Figma ComponentNodes via a two-pass build. Instances stay linked — ahead of every other plugin in the ecosystem.

🔤

Smart Font Fallback

Tries every weight and style variant before falling back. Shows you exactly which fonts were substituted and what they were replaced with.

🖼

CORS-Safe Image Pipeline

Images are fetched in the UI sandbox, converted to base64, and handed to Figma. Failed fetches show a visible gray placeholder — import never crashes silently.

📝

Rich Text Segments

Mixed-style text (bold spans, colored words, different sizes in one paragraph) maps to Figma's per-character range API — no flattening.

🔒

No Server Required

The plugin is pure client-side. Your designs never leave your machine. No API keys, no subscriptions, no rate limits.

📐

Min/Max Constraints

Responsive sizing constraints (minWidth, maxWidth, minHeight, maxHeight) are preserved on auto-layout frames and children.

⚠

Version Warnings

Files older than v2.9 format are detected and flagged before import — so you know upfront what may not transfer cleanly.

🎨

All Fill Types

Solid colors, image fills (with scale mode), linear, radial, and angular gradients all convert to their Figma equivalents.
