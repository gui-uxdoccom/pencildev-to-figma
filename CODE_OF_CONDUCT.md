# Code of Conduct

## The short version

Be kind. Be respectful. Help each other out.

## The slightly longer version

This project is open to everyone regardless of experience level, background, or identity. We're here to build a useful tool and have a good time doing it.

**Do:**
- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on what's best for the community and the project

**Don't:**
- Use sexualized language or imagery
- Troll, insult, or make derogatory comments
- Harass anyone, publicly or privately
- Publish others' private information without permission

## Enforcement

If someone is making the community unwelcoming, report it by opening an issue or contacting the maintainer directly. All complaints will be reviewed and handled appropriately.
